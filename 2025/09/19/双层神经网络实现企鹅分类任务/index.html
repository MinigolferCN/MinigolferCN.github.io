双层神经网络实现企鹅分类任务

在单层神经网络的基础之上增添一层隐藏层，就能得到了一般的多层神经网络，因而本文仅展示本项目与单层神经网络不同的地方。本文的代码基于李沐 《动手学深度学习》 https://zh.d2l.ai/chapter_multilayer-perceptrons/index.html
数据读取操作基于51cto用户mob649e8156b567的文章 https://blog.51cto.com/u_16175443/12175717

一、层数增加

增加层数要从几个方面来修改代码：

1.参数增加

hid = 8 #指定大小为8的隐藏层

W1 = torch.normal(0, 0.01, size=(input_size, hid), requires_grad=True)
W2 = torch.normal(0, 0.01, size=(hid, output_size), requires_grad=True)

b1 = torch.zeros(hid, requires_grad=True)
b2 = torch.zeros(3, requires_grad=True)

2.激活函数

引入多层神经网络后就需要使用激活函数了，否则将无法拟合非线性函数。这里选择了ReLU激活函数

def ReLU(X):
    return torch.maximum(torch.tensor(0.0), X)

3.网络连接修改

def net(X):
    hiddenlayer = ReLU(torch.matmul(X,W1) + b1)
    return softmax(torch.matmul(hiddenlayer.reshape((-1, W2.shape[0])),W2) + b2)

4.参数更新修改

def updater(batch_size):
    return d2l.sgd([W1,W2, b1,b2],lr,batch_size)

二、数据读取

由于上一个项目是针对图像的学习，这次修改为了对有具体含义的特征值的学习，需要修改数据读取的代码。
首先要创建一个继承自Dataset的类用于代替原先的ImageFolder函数：

class MyDataset(Dataset):
    def __init__(self, file):
        self.data_frame = file
        self.features = file.iloc[:, :-1].values.astype('float32')
        self.labels = file.iloc[:, -1].values

    def __len__(self):
        return len(self.data_frame)

    def __getitem__(self, idx):
        features = torch.tensor(self.features[idx])
        label = torch.tensor(self.labels[idx], dtype=torch.long)
        return features, label

然后修改data_loader函数

def data_loader(batch_size, root):  #root = "penguins_data.csv"
    file = pd.read_csv(root).dropna()
    file_len = len(file)
    file.insert(7, 'Specie', 0)

    for i in range(0, file_len):
        species = file.iloc[i].values[0]
        if species == "Adelie Penguin (Pygoscelis adeliae)":
            file.iat[i, 7] = 0
        elif species == "Chinstrap penguin (Pygoscelis antarctica)":
            file.iat[i, 7] = 1
        elif species == "Gentoo penguin (Pygoscelis papua)":
            file.iat[i, 7] = 2

    file = file.drop("Species", axis=1)
    file = file.sample(frac=1).reset_index(drop=True)

    train_dataset = MyDataset(file[0:301])
    test_dataset = MyDataset(file[301:])

    train_iter = DataLoader(train_dataset, batch_size=batch_size, shuffle=True, num_workers=0)
    test_iter = DataLoader(test_dataset, batch_size=batch_size, shuffle=False, num_workers=0)

    input_size = file.shape[1]-1
    output_size = 3

    print("类别: ",3)
    print("训练集大小: ",len(train_dataset), "测试集大小: ",len(test_dataset))

    return train_iter, test_iter, train_dataset, input_size, output_size

三、数据预处理

由于数据由原本的图片像素变为了真实值，在未进行数据预处理的情况下模型难以学习。

因此要先对数据进行预处理，这里选择进行数据标准化：

class MyDataset(Dataset):
    def __init__(self, file):
        self.data_frame = file
        # 数据标准化
        features = file.iloc[:, :-1].values.astype('float32')
        self.features = (features - features.mean(axis=0)) / (features.std(axis=0) + 1e-8)
        self.labels = file.iloc[:, -1].values

修改后模型正常开始训练：
